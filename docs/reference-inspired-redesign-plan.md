# Privorum Reference-Inspired Redesign Plan

## Goal

Use strong reference sites as inspiration for Privorum's next redesign steps without copying them directly.

The aim is to combine:

- enterprise trust
- AI workflow positioning
- backend/platform credibility
- stronger visual composition

## Redesign Plan

## Phase 1: Homepage

### Objective

Make the homepage feel immediately new and clearly positioned.

### Changes

1. Keep the dark hero, but refine it further.
2. Keep proof near the top.
3. Replace the current service block with AI-first service areas.
4. Improve the value section so it feels less like legacy cards.
5. Add one stronger "Why Privorum" trust section.

### Reference mapping

- Hero: GenieAI + Backend.AI
- Messaging: Automatic + Behest AI
- Service overview: Xano

## Phase 2: Services Page

### Objective

Make services clearly AI-first while keeping blockchain as a specialist capability.

### Changes

1. Keep the new AI-first service areas at the top.
2. Separate specialist blockchain pages into a lower section.
3. Improve visual distinction between strategic offers and legacy technical depth.

### Reference mapping

- Layout: Xano
- Messaging hierarchy: Behest AI

## Phase 3: Clients Page

### Objective

Turn the clients page into a stronger proof archive.

### Changes

1. Keep AI clients first.
2. Improve logo/name treatment further.
3. Reduce visual noise.
4. Introduce a better grouping rhythm without splitting the page into separate categories.

### Reference mapping

- Proof presentation: GenieAI
- layout cleanliness: enterprise software references

## Phase 4: Final Art Direction

### Objective

Remove the last traces of the old Hugo-theme feeling.

### Changes

1. Replace remaining old illustrations.
2. Add more custom system-driven shapes or compositions.
3. Refine spacing and section transitions.
4. Improve mobile composition for hero and proof sections.

### Reference mapping

- Visual atmosphere: Backend.AI
- composition confidence: GenieAI

## Concrete Deliverables

### Immediate deliverables

- stronger AI-first homepage
- AI-first service area cards
- more modern dark hero
- improved clients proof order

### Next deliverables

- upgraded proof styling
- custom visual assets or shapes
- stronger “Why Privorum” section

## Priority Order

1. Homepage composition and messaging
2. Service-area restructuring

## Success Criteria

The redesign should make visitors feel:

- this is a modern technical company
- Privorum works on AI workflows and backend systems now
- blockchain is still a capability, but not the entire identity
- the website is no longer a lightly modified Hugo theme
